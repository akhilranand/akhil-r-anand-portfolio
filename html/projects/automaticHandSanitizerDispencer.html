Sanitizer System

Zero-Contact Dispensing

Automated hand sanitizer dispenser using ultrasonic distance sensing and precise motor control for touch-free operation.

Live Demo

Project Overview

Developed during COVID-19 to provide reliable touchless sanitization. Detects hand proximity and dispenses exact sanitizer amount using servo-controlled pump.

Hand approaches → Ultrasonic trigger → Servo dispenses → Ready for next use

Key Features

  • Ultrasonic sensor (HC-SR04) for 5-20cm hand detection
  • Precise servo motor (SG90) pump control
  • Adjustable dispense volume via potentiometer
  • Low sanitizer level indicator with buzzer
  • Battery-powered portable design
  • Zero-touch operation (<1s response)

Tech Stack

  • Arduino Uno R3
  • HC-SR04 Ultrasonic Sensor
  • SG90 Micro Servo
  • 10kΩ Potentiometer
  • Active Buzzer
  • 9V Battery + Voltage Regulator
