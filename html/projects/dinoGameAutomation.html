Project Overview

Hardware bot that plays Chrome Dino game autonomously. LDR sensors detect shadow of approaching cacti/birds, triggering physical servo press on spacebar.

Obstacle shadow detected → LDR triggers → Servo jumps → Dino survives indefinitely

Key Features

  • 2x LDR sensors for obstacle shadow detection
  • High-torque servo for reliable spacebar press
  • Real-time light intensity thresholding
  • Configurable sensitivity via analog read
  • Unlimited gameplay (perfect score)
  • Compact 3D-printed enclosure

Tech Stack

  • Arduino Nano
  • 2x Light Dependent Resistors (LDR)
  • MG996R High-Torque Servo
  • 10kΩ Resistors (voltage divider)
  • 3D-printed mechanical linkage
  • USB power from PC
